Para executar o código, escrevemos alguns scripts em shell bash objetivando a facilitação.
Os scripts funcionam em ambientes linux com Bash.

Para compilar os códigos fontes, basta:

$ ./compile.sh

Para iniciar a aplicação, basta:

$ ./init.sh

Para gerar toda a documentação novamente, basta: 

$ ./generate_documentation.sh

Se houver erros de permissão, basta executar o seguinte comando na pasta raiz do projeto:

$ chmod +x *.sh
